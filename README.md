# LM Arena Time Analysis

Interactive dashboard tracking LM Arena leaderboard scores over time, with tabs for geographic groups, license types, and head-to-head org comparisons.

Live: https://geoalgo.github.io/lmarena-analysis/

## Setup

### 1. Clone the dataset

```bash
git clone https://huggingface.co/datasets/lmarena-ai/leaderboard-dataset
```

This clones ~45 MB of parquet files into `leaderboard-dataset/`. The file used is `leaderboard-dataset/text/full-00000-of-00001.parquet` (~906k rows, 2024–present).

### 2. Install dependencies

This project uses [uv](https://github.com/astral-sh/uv) for environment management.

```bash
uv sync
```

### 3. Process the data

```bash
uv run python3 process_data.py
```

This reads the parquet file, computes the best model per organization per date per category, and writes `lmarena-analysis/dashboard_data.json` (~9 MB).

### 4. Serve locally

```bash
uv run python3 -m http.server 8765 --bind 0.0.0.0
```

Then open `http://127.0.0.1:8765` in your browser.

> The dashboard fetches `dashboard_data.json` and `org_metadata.json` via relative paths, so it must be served — opening `index.html` directly from the filesystem will not work.

## Project structure

```
leaderboard-dataset/        # HuggingFace dataset (git clone, not committed)
lmarena-analysis/
  index.html                # Dashboard (single-page, Plotly.js)
  dashboard_data.json       # Generated by process_data.py
  org_metadata.json         # Organization metadata (origin, license mapping)
opensource_models.txt       # Curated list of truly open-source models
process_data.py             # Data processing script
```

## Dashboard tabs

| Tab | Description |
|-----|-------------|
| **Rating Over Time** | Arena score per organization, filterable by category and org |
| **🇺🇸 vs 🇨🇳 vs 🇪🇺 vs Rest** | Win probability of geographic group champion vs. other group champions |
| **Proprietary vs Open-weights vs Open-source** | Same metric grouped by license type |
| **Google vs Anthropic vs OpenAI vs xAI vs Meta** | Head-to-head win probability among top-5 US labs |

## Updating the data

Pull the latest dataset snapshot and re-run the processing script:

```bash
cd leaderboard-dataset && git pull && cd ..
uv run python3 process_data.py
```
